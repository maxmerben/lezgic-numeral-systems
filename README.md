# Numeral systems in Lezgic languages

This repository hosts the dataset and the R code for my article ___Numeral systems in Lezgic languages___, which is published in _Folia Linguistica_.

The article:
> Melenchenko, Maksim. (2024). _Numeral systems in Lezgic languages_. Folia Linguistica. https://doi.org/10.1515/flin-2024-2056
